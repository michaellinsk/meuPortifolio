Ultimos Projetos

Aryane Cardoso

landing page estratégica para divulgar o curso de automaquiagem da profissional Aryane Cardoso, com foco na captação de leads e gestão de inscritos.

Jogo do Numero secreto

Este projeto consiste em um jogo simples no qual o jogador precisa adivinhar um número secreto. A cada tentativa, o sistema fornece dicas sobre se o número é maior ou menor e indica se o número secreto é múltiplo de 2, 3, 5 ou 7. Com a integração da biblioteca ResponsiveVoice.js, as mensagens do jogo são lidas em voz alta, aumentando a interatividade com o usuário.

Decodificador de Texto

Uma aplicação web simples que permite ao usuário criptografar e descriptografar textos por meio de substituições de caracteres. É ideal para quem busca um projeto introdutório em JavaScript com interatividade direta. Além disso, oferece a funcionalidade de copiar o texto resultante para a área de transferência.

Previsão do Tempo

Uma aplicação web moderna e interativa para consultar a previsão do tempo em qualquer cidade do mundo. Usando a API do OpenWeatherMap, o projeto exibe informações essenciais como temperatura, umidade, velocidade do vento e descrição das condições climáticas. A aplicação foi construída com um design responsivo e intuitivo para garantir uma ótima experiência do usuário.
